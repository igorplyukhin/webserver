asynchronous multiprocessor webserver serving static requests

**Features**
- Asyncio realisation
- Keep-alive support
- Static requests
- Opened files descriptors cache
- Logging
- Virtual servers
- Configuration file
- Proxy pass
- CGI
- Bandwidth limit
- Regexp uri rewrite

**Reqiurements**
- Linux system
- `sudo apt install tree`
- python3.8
- `pip install --user lru-dict`

**Usage**
`python3.8 run_server.py`
